Quick Start

Up and running in minutes.

1

Clone the repository

git clone https://github.com/ashoksahoo/whatsapp-business-platform.git
cd whatsapp-business-platform
2

Configure your environment

Copy .env.example to .env and add your WhatsApp Business API credentials.

cp .env.example .env
# Set WHATSAPP_ACCESS_TOKEN and WHATSAPP_PHONE_NUMBER_ID
3

Run with Docker (easiest)

docker-compose up

Or run directly: go run cmd/server/main.go

4

Send your first message

curl -X POST http://localhost:8080/api/v1/messages \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"to": "+1234567890", "type": "text", "text": "Hello!"}'
